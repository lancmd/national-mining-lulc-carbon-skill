# 矿区类型规则

本文件规定不同矿区类型在土地利用分类、沉陷积水识别、碳储量计算和生态服务评价中的处理规则。

## 1. 煤矿区

### 1.1 普通煤矿区

适用地区：

- 山西煤矿区；
- 陕西煤矿区；
- 内蒙古煤矿区；
- 山东煤矿区；
- 贵州煤矿区；
- 其他地下开采煤矿区。

重点关注：

- 建设用地扩张；
- 工矿扰动地变化；
- 矸石山、排土场、采矿工业广场；
- 采煤沉陷区；
- 植被恢复。

分类建议：

- 水体；
- 建设用地；
- 耕地；
- 林地；
- 草地；
- 裸地/工矿用地。

### 1.2 高潜水位煤矿区

适用地区：

- 淮北矿区；
- 淮南矿区；
- 皖北煤矿区；
- 徐州矿区；
- 鲁西南煤矿区；
- 平原高潜水位采煤沉陷区。

重点关注：

- 沉陷积水；
- 自然水体；
- 水陆转化；
- 耕地损失；
- 湿地化演变；
- 沉陷积水碳库。

分类建议：

- 沉陷积水；
- 自然水体；
- 建设用地；
- 耕地；
- 林地；
- 草地；
- 裸地/工矿用地。

处理规则：

- 沉陷积水应结合矿区边界、沉陷区范围和历史影像识别；
- 不能只依靠 NDWI/MNDWI 自动区分沉陷积水和自然水体；
- 若研究碳储量，应单独计算沉陷积水复合碳库。

## 2. 金属矿区

适用对象：

- 铁矿；
- 铜矿；
- 金矿；
- 稀土矿；
- 铅锌矿；
- 其他金属矿。

重点关注：

- 露天采场；
- 排土场；
- 尾矿库；
- 工业场地；
- 裸地扩张；
- 植被恢复。

分类建议：

- 水体；
- 建设用地；
- 林地；
- 草地；
- 裸地/工矿用地；
- 尾矿库或采场可根据研究目标单独分类。

处理规则：

- 尾矿库可单独作为一类；
- 露天采场和排土场可合并为裸地/工矿用地；
- 若评价生态修复，应重点关注裸地向林地、草地的转化。

## 3. 非金属矿区

适用对象：

- 石灰岩矿；
- 砂石矿；
- 黏土矿；
- 建材矿山；
- 采石场。

重点关注：

- 露天采场；
- 裸露地表；
- 边坡恢复；
- 林草恢复；
- 建设扰动。

分类建议：

- 水体；
- 建设用地；
- 耕地；
- 林地；
- 草地；
- 裸地/工矿用地。

处理规则：

- 采场和裸露边坡可并入裸地/工矿用地；
- 修复后区域可根据植被覆盖情况划入林地或草地；
- 生态服务评价中应关注植被恢复和水土保持功能。

## 4. 废弃矿山修复区

适用对象：

- 关闭矿山；
- 历史遗留矿山；
- 废弃采坑；
- 生态修复工程区。

重点关注：

- 裸地减少；
- 林地增加；
- 草地增加；
- 水体变化；
- 建设用地整理；
- 修复工程生态效益。

分类建议：

- 水体；
- 建设用地；
- 耕地；
- 林地；
- 草地；
- 裸地/工矿用地。

处理规则：

- 修复工程前后应进行多期土地利用对比；
- 重点分析裸地/工矿用地向林地、草地、耕地和水体的转化；
- 可结合 NDVI、RSEI、碳储量和生态服务价值进行综合评价。
