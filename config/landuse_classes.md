# 土地利用分类体系

本文件用于规定全国矿区土地利用分类体系。不同矿区类型可根据研究目的选择普通六类体系或高潜水位煤矿区增强七类体系。

## 1. 普通矿区六类体系

适用于金属矿、非金属矿、普通煤矿、废弃矿山修复区和一般生态修复区。

| 编码 | 地类名称 | 说明 |
|---|---|---|
| 1 | 水体 | 河流、湖泊、水库、坑塘等 |
| 2 | 建设用地 | 城镇、村庄、道路、工业用地、光伏板等 |
| 3 | 耕地 | 水田、旱地、农田 |
| 4 | 林地 | 有林地、灌木林、人工林 |
| 5 | 草地 | 天然草地、人工草地、稀疏草地 |
| 6 | 裸地/工矿用地 | 采场、排土场、裸露地表、矿山扰动地 |

## 2. 高潜水位煤矿区增强七类体系

适用于采煤沉陷区、高潜水位矿区、沉陷积水明显发育区域。

| 编码 | 地类名称 | 说明 |
|---|---|---|
| 1 | 沉陷积水 | 由采煤沉陷、地表塌陷形成的积水区 |
| 2 | 自然水体 | 河流、湖泊、水库、天然坑塘等 |
| 3 | 建设用地 | 城镇、村庄、道路、工矿建设区、光伏板等 |
| 4 | 耕地 | 水田、旱地、农田 |
| 5 | 林地 | 有林地、灌木林、人工林 |
| 6 | 草地 | 草地、滩地植被、低覆盖植被 |
| 7 | 裸地/工矿用地 | 采场、排土场、矸石山、裸露扰动地 |

## 3. 分类合并规则

- 光伏板一般并入建设用地。
- 工业广场、矿井场地、道路、厂房并入建设用地。
- 露天采场、排土场、矸石山、裸露扰动地可并入裸地/工矿用地。
- 普通矿区不强制区分沉陷积水和自然水体。
- 高潜水位煤矿区应尽量区分沉陷积水和自然水体。
- 沉陷积水不能只依靠 NDWI 或 MNDWI 判断，应结合矿区边界、沉陷区范围、历史影像、地形低洼区和人工判读。
- 若分类精度不足，可先将沉陷积水和自然水体合并为水体，再在后处理中分离。

## 4. 面积换算规则

不同传感器空间分辨率不同，像元面积换算规则如下：

| 数据源 | 分辨率 | 面积换算 |
|---|---|---|
| Landsat 5/7/8/9 | 30 m | Count × 900 / 10000 = Count × 0.09 hm² |
| Sentinel-2 | 10 m | Count × 100 / 10000 = Count × 0.01 hm² |
| MODIS | 250 m | Count × 62500 / 10000 = Count × 6.25 hm² |

## 5. 输出要求

土地利用分类结果应至少输出：

- 分类栅格 GeoTIFF；
- 分类颜色表；
- 地类面积统计表；
- 精度评价结果；
- 若涉及多期变化，应输出土地利用转移矩阵。
